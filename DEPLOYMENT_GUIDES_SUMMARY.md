## 🚀 QUICK START (5 Minutes)

### Step 1: Upload Code to GitHub
```bash
cd "f:\research papers\isuru\new_deploy"
git init
git add .
git commit -m "Solar battery system"
git remote add origin https://github.com/YOUR_USERNAME/solar-battery.git
git push -u origin main
```

### Step 2: Create Railway Account
- Go to https://railway.app/
- Click "Start Free"
- Use GitHub to sign up

### Step 3: Deploy (One Click!)
- Click "New Project"
- Select "Deploy from GitHub"
- Choose "solar-battery" repository
- Click "Deploy Now"
- Wait 2-5 minutes

### Step 4: Access Your App
- Find your URL in Railway Dashboard
- Open `https://your-railway-url.railway.app`
- Done! 🎉

## ❓ FAQ

**Q: Do I need to install anything?**
A: No! Railway handles everything. Just push code to GitHub.

**Q: How much will it cost?**
A: $15-25/month for full deployment (pay-as-you-go).

**Q: Can I deploy for free?**
A: Railway has a free trial. After that, you pay for what you use.

**Q: How do I update my app?**
A: Just push to GitHub. If auto-deploy is on, Railway deploys automatically!

**Q: How do I view logs?**
A: Railway Dashboard → Logs tab (real-time output).

**Q: Can I add a custom domain?**
A: Yes! Railway supports custom domains ($10/month).

**Q: What if deployment fails?**
A: Check the "Deployments" tab in Railway dashboard for error logs.

**Q: Can I rollback if something breaks?**
A: Yes! Go to Deployments, click previous version, click Redeploy.
